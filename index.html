HTML

HTML (HyperText Markup Language) tags are the building blocks of web pages. They define the structure and content, including headings, paragraphs, links, images, and other elements. Tags are enclosed in angle brackets, like `< tag >`, and usually come in pairs, with an opening tag and a closing tag (``), to format and organize content for web browsers. some html tag is here

  • center tag
  • footer tag
  • main tag
  • div tag
  • li tag
  • form tag
  • label tag
  • input tag
  • button tag

CSS

CSS (Cascading Style Sheets) is a language used to style HTML documents. CSS allows you to control the layout, colors, fonts, and overall look of your web pages. By using selectors, properties, and values, you can apply styles to specific HTML elements. CSS enhances user experience by making websites visually appealing and easier to navigate. some css tag is here

  • margin
  • padding
  • background-color
  • font-family
  • font-size
  • text-align
  • text-transform
  • border
  • border-radius
